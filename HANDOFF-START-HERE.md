# 👋 Start here

You're the developer taking over the Seven Bison website. This is your entry point.

## Quick start (5 minutes)

```bash
git clone https://github.com/Szawar11/seven-bison-website.git
# or unpack the zip
cd seven-bison-website
npm install
npm run dev
# → http://localhost:3000
```

## Read these 3 documents — in this order

### 1️⃣ `DEVHANDOFF.md`  (5 min read)
**Stan projektu, what's done, what's pending, deploy instructions.**
Pełna mapa plików, design tokens, checklist prac.

### 2️⃣ `MEDIA-SPEC.md` (10 min read)
**Pełna lista wszystkich materiałów które dostarczy klient** — formaty, rozmiary, ścieżki, Vimeo vs upload guidance.
Każdy plik w UI ma już placeholder z dokładnym wymaganym formatem — placeholdery pokazują się na żywej stronie.

### 3️⃣ `DESIGN.master.md` (15 min read)
**Pełen design system** — kolory, typografia, spacing, motion, brand voice.
Czytaj tylko jeśli planujesz dodawać nowe komponenty.

## Twoja pierwsza praca

1. Otwórz projekt w VSCode
2. Sprawdź czy `npm run dev` startuje OK
3. Otwórz `localhost:3000` — zobacz placeholdery
4. Otwórz `MEDIA-SPEC.md` — przeczytaj sekcję "Vimeo vs upload"
5. Skontaktuj się z Szymonem (`hello@sevenbison.com`) po materiały — daj mu `MEDIA-SPEC.md` jako checklist
6. Gdy materiały dotrą → patrz "Etap 2" w `DEVHANDOFF.md`
